🚀 User Management API

🎯 Objetivo do Projeto

Este projeto foi desenvolvido para:

Praticar Spring Boot
Demonstrar arquitetura REST
Aplicar boas práticas de backend
Compor portfólio profissional

API REST desenvolvida com Spring Boot para gerenciamento de usuários, incluindo criação, listagem, busca e remoção.

📌 Sobre o Projeto

Esta API foi desenvolvida seguindo boas práticas de arquitetura em camadas, com separação de responsabilidades entre Controller, Service, Repository, DTO e Entity.

O objetivo do projeto é demonstrar habilidades em desenvolvimento backend com Java + Spring Boot, incluindo persistência de dados, organização de código e criação de endpoints REST.

🛠️ Tecnologias Utilizadas
Java 17
Spring Boot
Spring Data JPA
H2 Database
Maven
Lombok
REST API
